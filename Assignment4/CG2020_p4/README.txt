How to use: 
    Shader selection:
        press 1: use the gouraud shader
        press 2: use the phong shader

    Camera controls:
        Arrow keys: move camera in x and y axis
        I, J, K, L: orbit around mesh

    Light controls:
        W, S: move the current selected light in the y axis 
        A, D: move the current selected light in the x axis
        F, G: move the current selected light in the z axis

        C : create a new light with random color and intensity
        X : delete the current selected light
        Z : change selected light
    
    Mesh controls:
        + : add a Mesh (render one more)
        - : remove a Mesh (render one less)
        spacebar: rotate the meshes 

        Note: max 30 mesh and min 1 mesh
